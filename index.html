Styling the Web with CSS

Learn how to transform plain HTML into visually appealing web pages.

CSS Selectors

CSS selectors are used to target HTML elements for styling. The three most common selectors are:

  • Element selectors (e.g., p, h1)
  • Class selectors (e.g., .container)
  • ID selectors (e.g., #main-header)

Colors and Typography

CSS allows you to control colors using various formats:

  • Hex codes: #3498db
  • RGB: rgb(46, 204, 113)
  • Named colors: rebeccapurple

For typography, you can specify font families, sizes, weights, and more to create readable and attractive text.

Box Model

Every element is a box with:

  1. Content
  2. Padding
  3. Border
  4. Margin
